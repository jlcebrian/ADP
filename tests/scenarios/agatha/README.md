# Agatha's Folly walkthrough

Complete part-1 walkthrough with an ending-state assertion. Early in the
script, `@let 5 230` forces the grocery-lad timed event so the PAW More
prompt appears at the porch; `03-More.png` is anchored to the original
emulator capture in `fuse-reference/agatha-more.png` (pixel-identical) and
should not be casually re-recorded.

The game itself is not redistributable: scenario.json downloads the TZX
from Spectrum Computing (which has permission to host Zenobi titles) on
first use, verifying its hash.
